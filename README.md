# 仿生黎明 · Synthetic Dawn

> 薛定谔的喵 · 作品

一款赛博朋克风格的互动剧情游戏 HTML 单文件播放器。

## 🎮 在线试玩

**https://moompie666-bot.github.io/synthetic-dawn/**

## 📖 简介

在近未来世界，AI 仿生机器人与人类外形完全一致。他们在社会中寻求地位、权力、合法身份与自我定义。这一次 —— 由你决定他们的命运。

- 三条主线路线：凯（Kai）/ 莉拉（Lyra）/ 独行者（Wanderer）
- 五种结局：仿生领袖 / 人类守护 / 超越 / 共存 / 悲剧
- 支持存档、读档、CG 画廊、对话回溯、自动播放

## 📱 支持设备

- 💻 桌面端（Chrome / Edge / Firefox / Safari）
- 📱 手机端（iOS Safari / Android Chrome，响应式适配）
- 🖥️ 平板端（iPad / Android Tablet）

## 🛠️ 技术栈

- 原生 HTML5 + CSS3 + 纯 JavaScript（无框架依赖）
- 单文件部署，离线可用
- 剧本由 Ren'Py 源码经 `export_story_html.py` 编译而来
